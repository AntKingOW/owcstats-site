# Asset Slots

Add local images using these file names so the UI can pick them up automatically.

## Team Logos

- Folder: `assets/teams`
- File name: `{team-slug}.png`
- Example: `team-falcons.png`

## Map Backgrounds

- Folder: `assets/maps`
- File name: `{map-slug}.webp`
- Fallbacks also work: `.jpg`, `.png`
- Example: `lijiang-tower.webp`

## Hero Portraits

- Folder: `assets/heroes`
- File name: `{hero-slug}.png`
- Example: `brigitte.png`

## Slug Rule

- Lowercase
- Replace spaces and punctuation with `-`
- Examples:
  - `ZETA DIVISION` -> `zeta-division`
  - `Team Falcons` -> `team-falcons`
  - `D.Va` -> `d-va`
  - `Soldier: 76` -> `soldier-76`
  - `Lijiang Tower` -> `lijiang-tower`
