# BanCalculatorOW

BanCalculatorOW is a separate web project for building an Overwatch ban calculator and eventually deploying it online.

## Goals

- Let users input team bans or draft situations
- Calculate recommended ban outcomes or ban states
- Provide a clean web UI
- Prepare for public deployment later
