# BanCalculatorOW Project Brief

## Purpose

BanCalculatorOW is a web-based Overwatch ban calculator project.

## Direction

- Separate project from OWCStats
- Focus on ban calculation and ban-related UX
- Design-first web approach
- Deployable structure from the beginning

## Proposed Early Scope

1. Define inputs
   - Team A
   - Team B
   - Initial ban
   - Follow-up ban
   - Remaining hero pool

2. Define outputs
   - Current banned heroes
   - Available heroes after bans
   - Role-by-role remaining pool
   - Optional recommendation or warning states

3. Web goals
   - Fast and simple interface
   - Mobile-friendly layout
   - Public deployment later

## Immediate Next Task

Create the first functional web app scaffold and define the exact ban logic.
